# Aamir Ali Master Goldsmith

A jewellery website created as a tribute to **Aamir Ali Master Goldsmith**, celebrating a legacy of timeless craftsmanship, tradition, and elegance.

## Project Overview

This project showcases an elegant jewellery website with a clean and modern design. It highlights handcrafted jewellery collections and reflects the legacy of Aamir Ali Master Goldsmith.

## Technologies Used

* HTML5
* Tailwind CSS

## Features

* Clean and modern user interface
* Hero banner
* Jewellery collection page
* About section
* Contact form
* Newsletter subscription
* Well-organized code structure

## Purpose

This project is created as a tribute while also serving as a front-end practice project to improve my HTML and Tailwind CSS skills.
